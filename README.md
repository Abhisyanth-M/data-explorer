Smart Data Explorer Dashboard
Overview

Smart Data Explorer Dashboard is a simple and interactive web application built using Streamlit and Pandas.
It allows users to upload CSV files and explore their data without writing code.

Features
Upload CSV files
Select any column dynamically
Filter data based on selected values
View filtered dataset instantly
Display full dataset preview
Show basic statistical summary
Technologies Used
Python
Streamlit
Pandas
How It Works
User uploads a CSV file
The app reads the file using Pandas
User selects a column from dropdown
User selects a value from that column
App filters and displays matching data
Full dataset and summary statistics are shown
How to Run Locally
pip install streamlit pandas
python -m streamlit run app.py
